# PING!

A tiny arcade game in pygame. One paddle, one ball, don't let it drop, chase the high score.

<img src="assets/Background.png" width="480" alt="PING background art">

## The details

- 1280x720, vsync, sound effects
- High score persists between runs (saved next to the exe when packaged)
- Packaged to a Windows exe with PyInstaller

## Play it

```bash
pip install pygame
python Game.py
```

Escape quits. That's the whole manual.
